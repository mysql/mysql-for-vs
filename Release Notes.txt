Connector/Net 6.0  Release Notes
------------------------------------

Welcome to the release notes for Connector/Net 6.0

This release introduces initial support for entity framework.  This is our beta release.  We know of 
some minor issues with the new visual studio integration and these will be corrected for our GA release.
Please let us know about any issues you have and we'll address them quickly.

We hope you enjoy this release and encourage you to test this release with your applications
and let us know any problems you find.
