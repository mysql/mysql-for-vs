MySQL Connector/Net
Compatibility
MySQL compatibility

Connector/Net fully supports all GA versions of MySQL from 3.23 to 5.0. Connector/Net also supports the latest pre-release versions of MySQL 5.1 and 6.0.

Framework compatibility

Connector/Net is tested and works with .NET Frameworks 1.1 and 2.0 and .NET Compact Framework 2.0. It is also compatible with Mono.

Developer Tools

Connector/Net currently provides for tight integration with Visual Studio 2005 Standard and higher. This includes DDEX integration.
